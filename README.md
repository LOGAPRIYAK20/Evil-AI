
NightGuard - AI-Powered Insider Threat Investigation Platform
Overview

NightGuard is a cybersecurity project that detects potential insider threats within an organization.

The system monitors employee activities, identifies suspicious behavior, calculates risk scores, generates investigation reports, and visualizes threats through an interactive dashboard.

-----------------------------------


FEATURES


Large Download Detection

USB Usage Detection

Sensitive File Access Detection

Unusual Working Hours Detection

Behavioral Baseline Analysis

Risk Scoring System

Automated Alert Generation

AI Investigation Reports

Evidence Report Generation

Interactive Streamlit Dashboard

-------------------------------------------------------------------------

HOW IT WORKS

Employee activity logs are collected.

The system analyzes user behavior.

Suspicious activities are detected.

Risk scores are calculated.

Threat levels are assigned.

Investigation reports are generated.

Results are displayed on the dashboard.
